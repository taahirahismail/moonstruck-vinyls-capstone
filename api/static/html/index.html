Routes Descriptions!

HTTP Methods Path Description
GET '/' Home (current) page - all routes and their descriptions.
GET '/users' fetch details of all users.
GET '/user/:id' fetch details of single user identified by userID.
POST '/register' add details of a new user.
POST '/login' allow an existing user to access the site using their credentials.
PUT / PATCH '/user/:id' update the details of a single user.
DELETE '/user/:id' delete a single user from records.
GET '/products' fetch details of all products.
GET '/product/:id' fetch details of single product.
POST '/product' add details of a new product.
PUT / PATCH '/product/:id' update details of a single product.
DELETE '/product/:id' delete a single product from records.
GET '/orders' fetch all orders.
GET '/user/:id/carts' fetch cart of single user (identified by userID).
POST '/user/:id/cart' add products to cart.
PUT / PATCH '/user/:id/cart/:id' update products in cart.
DELETE '/user/:id/cart' clear cart of all products.
DELETE '/user/:id/cart/:id' remove single product from cart (identified by prodID).
